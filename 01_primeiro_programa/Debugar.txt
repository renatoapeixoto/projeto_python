Passo a Passo para Usar o Debugger no VS Code
1 - Abra o Código no VS Code Primeiro, abra seu código Python no VS Code. 
É como abrir uma receita de bolo que você quer concordar.

2 -Defina um Ponto de Parada (Breakpoint) Um breakpoint é como um ponto de 
parada no seu código. Imagine que você quer parar em uma linha específica para 
ver o que está acontecendo naquele ponto. Para fazer isso:
    Clique na margem esquerda da linha onde você quer parar, e um pontinho 
    vermelho vai aparecer ali. Esse é o seu breakpoint .

3 - Inicie a Depuração Agora que você colocou um ponto de parada, inicie 
uma depuração.
    No VS Code, pressione F5ou clique no botão verde de "Iniciar Depuração". 
    Isso vai começar a executar o código até chegar no ponto de parada.

4 - Veja o que está acontecendo Quando o código para no breakpoint , o VS Code vai 
mostrar o que está acontecendo nessa linha. Você pode ver o valor das variáveis 
​​e o que elas estão armazenadas naquele momento. Por exemplo, se você tiver uma
chamada variável idade, você poderá ver quantos anos ela está armazenando.

5 - Passo a Passo no Código Agora você pode ir linha por linha para ver o que o
 código está fazendo. Você pode usar:
    F10 para avançar para a próxima linha sem entrar em detalhes das funções.
    F11 para entrar em uma função e ver o que ela está fazendo por dentro.
    Shift + F11 para sair da função e voltar para o código principal.

6 - Verificando Variáveis​
​Quando o código está parado, você pode ver o valor de 
todas as variáveis ​​no painel lateral. Isso é útil para entender se os valores 
estão corretos. Se você quiser ver algo específico, pode adicionar uma variável 
no painel "Watch" (Observação) e acompanhar o valor dela enquanto o código roda.

7 - Continuar ou Parar a Depuração
    Se você quiser continuar o código até o próximo breakpoint , aperte F5.
    Se você deseja interromper a depuração completamente, aperte Shift + F5.
